#recast.ly

## Stack

Built using Node, Express, React. 

####Future Deployment: 
Docker, AWS EC2, AWS Lambda.

####Build tools:
NPM scripting, Webpack, ESlint.

####Testing: 
Mocha, Chai.

### Installing Dependencies

From within the root directory:
```sh
npm install
```

### Usage

Run the following command for a static build
```sh
npm start
```
